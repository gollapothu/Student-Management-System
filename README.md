# Student Management System

A Python-based Student Management System with JSON file storage.

## Features

* Add Student
* View Students
* Search Student
* Update Student
* Delete Student
* Add Marks
* Add Attendance
* Grade Calculation
* Student Report Generation
* JSON Data Storage

## Technologies Used

* Python
* JSON
* File Handling

## How to Run

```bash
python student_management.py
```
